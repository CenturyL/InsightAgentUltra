                <blockquote><p></p><h3>项目概况</h3>          数据治理及应用（一期）建设项目 招标项目的潜在投标人应在北京市政府采购电子交易平台获取招标文件，并于2025-07-02 09:30（北京时间）前递交投标文件。          <p></p></blockquote> <p><strong>一、项目基本情况</strong></p> <p>项目编号：11000025210200139384-XM001</p> <p>项目名称：数据治理及应用（一期）建设项目</p> <p>预算金额：143 万元（人民币）</p> <p>最高限价：143 万元（人民币）</p> <p>采购需求：</p><p></p><p><p>详见附件：招标公告原文。</p></p> <p>合同履行期限：（服务期限）：项目交付期限为2025年11月20日前</p> <p>本项目不接受联合体投标。</p> <p><strong>二、申请人的资格要求：</strong></p> <p>1.满足《中华人民共和国政府采购法》第二十二条规定；</p> <p>2.落实政府采购政策需满足的资格要求：</p> <p><p>2.1 中小企业政策：本项目专门面向中小企业采购。</p><p>2.2 其它落实政府采购政策的资格要求： 无 </p></p> <p>3.本项目的特定资格要求：</p> <p><p>3.1本项目是否接受分支机构参与投标：否；</p><p>3.2本项目是否属于政府购买服务：否；</p><p>3.3 其他特定资格要求：无。</p></p> <p><strong>三、获取招标文件</strong></p> <p>时间：2025-06-11 至 2025-06-18          ，每天上午09:00至11:30，下午13:30至16:30（北京时间，法定节假日除外）</p> <p>地点：北京市政府采购电子交易平台</p> <p>方式：</p> <p><p>投标人持CA数字认证证书（北京一证通数字证书）登录北京市政府采购电子交易平台获取电子版招标文件</p></p> <p>售价：￥0 元，本公告包含的招标文件售价总和</p> <p><strong>四、提交投标文件截止时间、开标时间和地点</strong></p><p> 2025-07-02 09:30（北京时间）</p> <p>地点： 中招国际招标有限公司会议室（具体会议室详见中招公司6层大屏幕）</p> <p><strong>五、公告期限</strong></p> <p>自本公告发布之日起5个工作日。</p> <p><strong>六、其他补充事宜</strong></p> <p><p>6.1.本项目的招标公告仅同时在中国政府采购网（www.ccgp.gov.cn）、北京市政府采购网（www.ccgp-beijing.gov.cn）、北京市政府采购电子交易平台上发布。</p><p>6.2.本项目需要落实的政府采购政策：政府强制采购的节能产品、环境标志产品、政府采购促进中小企业发展、政府采购项目支持监狱企业发展、政府采购促进残疾人就业、政府采购信用担保等相关政策详见招标文件。</p><p>6.3.本项目采用半流程电子化招标方式，请投标人认真学习北京市政府采购电子交易平台发布的相关操作手册，办理CA认证证书、进行北京市政府采购电子交易平台注册绑定，并认真核实数字认证证书情况确认是否符合本项目电子投标要求。</p><p>CA认证证书服务热线 010-58511086</p><p>技术支持服务热线    010-86483801    13669922829</p><p>6.3.1办理CA认证证书（北京一证通数字证书）</p><p>投标人登录北京市政府采购电子交易平台（http://zbcg-bjzc.zhongcy.cn/bjczj-portal-site/index.html）查阅 “用户指南”—“操作指南”—“市场主体CA办理操作流程指引”，按照程序要求办理。</p><p>6.3.2电子交易平台注册</p><p>投标人登录北京市政府采购电子交易平台“用户指南”—“操作指南”—“市场主体注册入库操作流程指引”进行自助注册绑定。</p><p>6.3.3驱动、客户端下载</p><p>投标人登录北京市政府采购电子交易平台“用户指南”—“工具下载”—“招标采购系统文件驱动安装包”下载相关驱动。</p><p>6.3.4 获取电子招标文件</p><p>投标人持CA数字认证证书（北京一证通数字证书）登录北京市政府采购电子交易平台获取电子版招标文件。未在规定期限内通过电子交易平台获取招标文件的投标无效。</p><p>6.3.5编制投标文件</p><p>投标文件由“资格证明文件”和“商务技术文件”组成。投标人应完整地按照招标文件第七章 投标文件格式中提供的投标文件格式及要求编写投标文件。投标文件中资格审查和符合性审查涉及的事项不满足招标文件要求的，其投标将被认定为投标无效。上述文件应按照招标文件的规定签署和盖公章或经公章授权的其他单位章（以下统称公章）。采用公章授权方式的，应当在投标文件资格证明文件中附公章授权书（格式自定）。</p><p>6.3.6递交投标文件</p><p>投标人应于投标截止时间前在接收投标文件的地点递交投标文件，须递交正本：1份、副本：4份；除上述文件外，还须密封递交投标文件电子文档（仅限U盘）：PDF及WORD格式各1份。其中：PDF格式应为投标文件正本加盖公章后的扫描件。</p><p>电子文档要求单独密封于一信封，并在信封上标明“投标文件电子版”字样，在投标时单独递交。</p><p>6.3.7开标</p><p>投标人于开标时间在开标地点进行开标。</p></p> <p><strong>七、对本次招标提出询问，请按以下方式联系。</strong></p> <p>1.采购人信息</p> <p>名 称：北京商贸学校（北京市SPF技术研究与推广中心）　　　　　</p> <p>地址：北京市大兴区西红门南西路14号　　　　　　　　</p> <p>联系方式：曹旭,（010）60223907　　　　　　</p> <p>2.采购代理机构信息</p> <p>名 称：中招国际招标有限公司　　　　　　　　　　　　</p> <p>地　址：北京市海淀区学院南路62号中关村资本大厦601A室　　　　　　　　　　　　</p> <p>联系方式：钱卉卉、李振、刘雨婷、韩颖、杨凡，（010）62108186、62108246、62108001、61954087　　　　　　　　　　　　</p> <p>3.项目联系方式</p> <p>项目联系人：钱卉卉、李振、刘雨婷、韩颖、杨凡</p> <p>电　话：　　（010）62108186、62108246、62108001、61954087</p>
