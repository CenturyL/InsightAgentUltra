 浦东新区各相关单位：
 
   根据上海市知识产权局《关于开展专利密集型产品支持政策申报工作的通知》（沪知局促〔2025〕4号），市知识产权局将对本市首次获得认定的专利密集型产品予以每件1万元的奖励。
 
   请浦东新区2023年（23件）、2024年（31件）已被认定有专利密集型产品的单位，填写附件《单位填报信息表》，并将盖章后的扫描件发电子邮件至wangqn@pudong.gov.cn，截止时间2025年3月10日。
